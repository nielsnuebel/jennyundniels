Introduction

First of all thank you for purchasing the template. If you have any questions that are beyond the scope of this help file, please feel free to email at support@choco-pixel.com. Any improvement or suggestion for some new feature will also be welcomed.

The documentation bellow is divided into sections for each version of template, starting with common instructions.

Common settings

Modifying countdown timer

  1. Open js/init.js
  2. In section Countdown, change the date as year/month/day

Example: 20th of October 2015 would look like: 2015/10/20

Modifying contact e-mail

  1. Open php/include/settings.php
  2. Enter your desired e-mail address $to = 'YOUR-EMAIL-ADDRESS';

Example: contact@mydomain.com would look like: $to = 'contact@mydomain.com';

E-mail subscription

Setting up database

  1. Open phpMyAdmin

  6. Copy the following into the field:

    CREATE TABLE `signups` (
`signups_id` int(10) NOT NULL AUTO_INCREMENT,
`signup_email_address` varchar(250) DEFAULT NULL,
`signup_timestamp` timestamp,
PRIMARY KEY (`signups_id`) 
) ENGINE=MyISAM AUTO_INCREMENT=1 DEFAULT CHARSET=utf8;

    Click Go and the database is ready.

Modifying credentials

  1. Open php/include/settings.php
  2. Edit database credentials: 
    // Database
$host = "YOUR-HOST-NAME";
$username = "YOUR-HOST-USER";
$password = "YOUR-HOST-PASSWORD";
$database = "TEMPLATE-DATABASE";

Setting up MailChimp

  1. If you don't have an account, go to login.mailchimp.com/signup and enter your credentials, otherwise go to the next step
  2. If you already have a list set up, go to the next step. On MailChimp dashboard click on Lists and in top right corner click on Create list and fill the List details
  3. Click again on Lists and click from dropdown menu Settings
  4. At the bottom of page, copy Unique id for list and paste in php/include/settings.php file 
    // MailChimp
$listID = "LIST-ID";
  5. Next go to http://admin.mailchimp.com/account/api and click on Create a Key button and copy and paste into same file as in above step (php/include/settings.php) and MailChimp integration is ready 
    $APIKey = "API-KEY";

Choosing e-mail subscription variant

E-mail subscription could be realised with database storage or with MailChimp integration.

  1. Open php/include/settings.php
  2. Set variable $useMailChimp value YES if you want MailChimp integration or NO if you want database storage

Example: Enabling MailChimp intergration would look like:

// Variant
$useMailChimp = "YES";

Background template

Modifying background image

  1. Open js/init.js
  2. In section Background, replace the path to image, $.backstretch("PATH/TO/IMAGE");

Example: for image myBackgroundImage.jpg would look like: $.backstretch("PATH/TO/myBackgroundImage.jpg");

Slideshow template

Modifying slideshow images

  1. Open js/init.js
  2. In section Background, replace the paths to images, 
    $.backstretch([
	"PATH/TO/IMAGE-1",
	"PATH/TO/IMAGE-2",
	"PATH/TO/IMAGE-3",
	...
], {duration: 3800, fade: 1500});
  3. The duration denotes how much time the image stays visible before fadeing out.

Example: for images myBackgroundImage-1.jpg, myBackgroundImage-2.jpg, myBackgroundImage-3.jpg would look like:

$.backstretch([
	"PATH/TO/myBackgroundImage-1.jpg",
	"PATH/TO/myBackgroundImage-2.jpg",
	"PATH/TO/myBackgroundImage-3.jpg"
], {duration: 3800, fade: 1500});

Video template

Adding youtube video

Warning! The video will not play, if the theme is not running on the server.
  1. Open js/init.js
  2. In section Background video, replace the id with your desired video videoId:'YOUTUBE-VIDEO-ID'

Example: for Youtube video with https://www.youtube.com/watch?v=ab0TSkLe-E0 url would look like: videoId:'ab0TSkLe-E0'

Adding local video

  1. Open js/init.js
  2. In section Background video, replace the paths with your desired video, for each video type and poster image.

Example: for video with MYVIDEO filename would look like: 

$("body").vide({
    mp4: 'PATH-TO/MYVIDEO.mp4',
    webm: 'PATH-TO/MYVIDEO.webm',
    ogv: 'PATH-TO/MYVIDEO.ogv',
    poster: 'PATH-TO/MYVIDEO.jpg'
});
